# Mix

## Frontend

### Requirements

Run `npm install` to install packages.

### Run

To start run:

`
$ cd frontend
$ ng serve --host 0.0.0.0
`

## Backend

### Requirements

Do `pip3 install requirements.txt` to install requirements.

### Run

To start run:

`
$ cd backend
$ python3 manage.py runserver 0.0.0.0:8001
`
